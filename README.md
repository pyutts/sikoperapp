# SIKOPER [Sistem Informasi Koperasi]

![Java](https://img.shields.io/badge/Java-11%2B-blue)
![License](https://img.shields.io/github/license/pyutts/sikoperapp)

## Deskripsi
SIKOPER atau disebut dengan Sistem Informasi Koperasi, ialah sebuah sistem informasi yang bisa digunakan untuk merekap anggota, pencatatan, dan rekapan uang secara transparan dan tersistem

## Fitur Utama
- **Fitur Menambahkan Anggota**
- **Fitur Simpanan**
- **Fitur Pinjaman**
- **Fitur Rekapan/Laporan**

## Prasyarat
Pastikan Anda sudah menginstal:
- Java 11 atau lebih baru
- Pastikan Terinstal Java FX SDK
- Terinstall MySQL

## Instalasi
1. Clone repositori ini:
    ```bash
    git clone https://github.com/username/nama-repo.git
    cd repo-name
    ```
2. Bangun proyek dengan Maven (jika menggunakan Maven):
    ```bash
    mvn clean install
    ```
3. Konfigurasi file `application.properties` atau file konfigurasi lain sesuai dengan kebutuhan Anda (misalnya, konfigurasi database).

4. Jalankan aplikasi:
    ```bash
    java -jar target/nama-aplikasi.java
    ```
